# JeOS Gateway
OpenVPN gateway based on SLES 12 JeOS

## Installation

1. Install SLE 12 SP1 JeOS
2. Register your machine with

    SUSEconnect -e john.doe@example.com -r <your_key>

3. Install software with

    zypper in openvpn dnsmasq
    foobar
